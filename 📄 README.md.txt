# WordPress Resume Blog Creator

This project publishes a resume-style blog to WordPress using the WordPress REST API and Python.

## Features

- Posts structured resume sections: About, Projects, Coaching, Poems, etc.
- Uses `blog_content.json` to feed dynamic sections.
- Can be reused for any personal blog.

## How to Use

1. Set up WordPress + App Password.
2. Edit `blog_content.json` with your content.
3. Run `main.py`.
